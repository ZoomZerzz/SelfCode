## Подключение бота

Прописывайте данные команды в termux, по очереди.

```bash
  pkg install git -y 
```

```bash
  pkg install python -y 
```

```bash
  git clone https://github.com/ZoomZerzz/SelfCode
```

```bash
  pip install googletrans
  pip install pyrogram
  pip install random
```
```bash
  cd SelfCode
```
```bash
  python index.py
```

Прописывайте данные команды в termux, по очереди.

```bash
  pkg install git -y 
```

```bash
  pkg install python -y 
```

```bash
  git clone https://github.com/ZoomZerzz/SelfCode
```
```bash
  pip install -r requirements.txt
```
```bash
  cd SelfCode
```
```bash
  python index.py
```
